Class MinifyHTML

Compress HTML

This is a heavy regex-based removal of whitespace, unnecessary comments and tokens. IE conditional comments are preserved. There are also options to have STYLE and SCRIPT blocks compressed by callback functions.

A test suite is available.

Package: Minify
Author: Stephen Clay <steve@mrclay.org>
